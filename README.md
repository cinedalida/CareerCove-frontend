# CareerCove-frontend

This is the frontend [COPY] of the CareerCove application built with React, Vite, CSS and TailwindCSS.

## Installation Guide

### 1. Clone the repository

```bash
git clone https://github.com/your-username/CareerCove-Frontend.git
```

```bash
cd my-app
```

2. Install dependencies

```bash
npm install
```

3. Run the development server

```bash
npm run dev
```
