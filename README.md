# CareerCove-frontend


This is the frontend of the CareerCove application built with React, Vite, and TailwindCSS.

## Installation Guide

### 1. Clone the repository
```bash
git clone https://github.com/your-username/CareerCove-Frontend.git
```
```bash
cd CareerCove-Frontend
cd my-app
```
2. Install dependencies
```bash
npm install
```
3. Run the development server
```bash
npm run dev
```
The app will be available at:
```bash
http://localhost:5173/
```
4. Build for production
```bash
npm run build
```
